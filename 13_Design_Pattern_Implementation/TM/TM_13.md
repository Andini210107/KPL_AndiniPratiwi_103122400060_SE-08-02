## Soal
Jelaskan dengan kemampuanmu apa itu event delegation dalam design pattern JavaScript. Tidak ada batas bobot kata dalam menjawab tugas ini, tetapi penilaian akan bergantung dari sepaham apa dan sebagus apa kamu menyajikan jawabanmu.

## Jawaban
Menurut saya, **event delegation** adalah teknik dalam JavaScript untuk menangani banyak event dengan lebih efisien. Daripada memasang event listener pada setiap elemen, kita cukup memasangnya pada elemen induk (parent). Nanti ketika salah satu elemen anak diklik atau menjalankan event tertentu, parent dapat mendeteksinya melalui mekanisme **event bubbling**.

Misalnya ada banyak tombol dalam satu container. Jika setiap tombol diberi event listener, kode akan menjadi lebih panjang dan kurang efisien. Dengan event delegation, cukup satu event listener pada container, lalu menggunakan `event.target` untuk mengetahui tombol mana yang ditekan.

Kelebihan event delegation adalah penggunaan memori menjadi lebih hemat karena jumlah event listener lebih sedikit. Selain itu, teknik ini juga memudahkan saat ada elemen baru yang ditambahkan secara dinamis karena tidak perlu membuat event listener baru untuk elemen tersebut.

Jadi, dapat disimpulkan bahwa event delegation merupakan cara yang lebih praktis dan efisien untuk mengelola event pada banyak elemen dalam JavaScript.
